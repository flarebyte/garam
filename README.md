> Local web service to store  keywords and metadata for files and urls


## Install

```sh
$ npm install --save garam
```


## Usage

```js
var garam = require('garam');

garam('Rainbow');
```

```sh
$ npm install --global garam
$ garam --help
```

```sh
# creates a browser.js
$ npm run browser
```
